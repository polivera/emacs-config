Suggested bootstrap order

evil-mode — Vim keybindings, do this first or you'll hate everything
use-package — package management (built-in since Emacs 29)
vertico + orderless + consult — modern completion UI, the current best-practice stack
corfu — in-buffer completion (replaces company)
eglot — LSP, built-in since 29, zero dependencies
magit — Git, install this early, it's life-changing
org-mode — built-in, just configure it
evil-org — makes org navigation feel natural with Vim keys
